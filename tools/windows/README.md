# Windows Operations Scripts

Bu klasör Windows operasyon scriptlerini içerir.

- `DB_AC.bat`: pgAdmin / DBeaver açma yardımcı arayüzü
- `DB_DOGRULA.bat`: veritabanı temizleme SQL'ini `DRY-RUN` doğrular
- `DEV_SERVER_RESTART.bat`: dev sunucuyu restart eder
- `MASAUSTU_AC.bat`: desktop shell geliştirme açılışı
- `MASAUSTU_BUILD.bat`: Windows desktop paket üretimi
- `PROD_ONIZLEME.bat`: production benzeri web/API önizleme
- `VERITABANI_TEMIZLE.bat`: tüm uygulama verilerini temizler
- `YEDEK_AL.bat`: veritabanı + uploads yedeği alır
- `YEDEKTEN_GERI_YUKLE.bat`: yedekten geri yükleme yapar

Not: Scriptler proje kökünü çalışma dizini olarak ayarlayacak şekilde güncellenmiştir, bu nedenle bu klasörden veya proje kökünden çağrılabilir.
